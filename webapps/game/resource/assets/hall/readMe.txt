大厅配置说明

buildingPosition 建筑位置,当前使用EUI--已废弃

logo 每一个地图等级对应的 logo位置

dinnerPosition 每一个地图等级对应的餐厅位置

barPosition 每一个地图等级对应的吧台位置

seatPosition 每一个地图等级对应的坐下位置

roadBridge_enter 每一个地图等级对应的人物进入餐厅时走到位置的桥接点

roadBridge_exit 每一个地图等级对应的人物走出餐厅时从位置到走出路径的桥接点

articlePosition 每一个地图等级对应的摆件坐标，需要注意的是 43，树，有两个坐标 ，44，餐桌，多个坐标，45，椅子，多个坐标，代码里有对应，会创建多个

enterPointBase 进入的基础路径，每一个地图等级都一样

outPointBase 走出的基础路径，每一个地图等级都一样

waiting 进入大厅时宾客在餐厅外的等待位置，每一个地图等级都一样

waitor 迎宾的位置，目前每一个地图等级都一样

chefRoad 每一个地图等级对应的厨师进入厨师位置的桥接路径，注意这里厨师进入餐厅跟走出餐厅都使用了同一个桥接路径
